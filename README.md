## 🧾 Overview

A fully responsive personal portfolio website built using HTML, CSS, and JavaScript, with Bootstrap ensuring seamless adaptability across all devices. The website presents a clean and structured showcase of my skills, projects, and professional journey, focusing on both functionality and visual appeal.

---

## ✨ Key Features

* Responsive design optimized for mobile, tablet, and desktop
* Clean, modern UI with smooth animations
* Interactive project showcase section
* Downloadable resume integration
* Functional contact form
* Dark/Light mode toggle
* Optimized performance for fast loading
* Accessibility-friendly and user-centric design

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Framework:** Bootstrap

---

## 🎯 Purpose

This portfolio is designed to effectively present my technical expertise, projects, and experience in a professional format. It serves as a central platform for recruiters, clients, and collaborators to explore my work and capabilities.
